# Forgeuslously-Optimized-Legacy-
Best Performance in Minecraft 1.12.2 Without Optifine! (based on Fabulously Optimized)

Forgeulously Optimized For 1.12.2!
An (unofficial) port of Forgeulously Optimized, unfortunately, most of Optifine's features are not available for 1.12.2, there's no support for shaders or EMF and ETF (necessary for texture packs like Fresh Animations)... but even so, it has the best performance thanks to Relictium, the (almost) only port of Sodium for 1.12.2!
